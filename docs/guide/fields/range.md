# Relation

A range slider.

## Example

```php
[
    'id' => 'range',
    'min' => 1,
    'max' => 10,
    'step' => 1,
    'type' => 'range',
    'title' => 'Range',
    'placeholder' => 'Range',
    'hint' => 'Chose a number between 1 and 10',
    'width' => 12
],
```

## Methods

| Method  | Description                                              |
| ------- | -------------------------------------------------------- |
| `title` | The title description for this field.                    |
| `hint`  | A short hint that should describe how to use the field.` |
| `cols`  | Cols of the field.                                       |
| `min`   | Minimum value.                                           |
| `max`   | Maximum value.                                           |
| `step`  | Steps. (default: 1)                                      |
