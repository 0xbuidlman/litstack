# DateTime

A DateTime-Picker.

Example

```php
[
    'id' => 'date',
    'type' => 'dt',
    'formatted' => 'l',
    'title' => 'Date',
    'placeholder' => 'Date',
    'hint' => 'Chose a date',
    'width' => 12
],
```

## Formats

| key  | Format                           |
| ---- | -------------------------------- |
| LT   | 2:15 PM                          |
| LTS  | 2:15:52 PM                       |
| L    | 12/08/2019                       |
| l    | 12/8/2019                        |
| LL   | December 8, 2019                 |
| ll   | Dec 8, 2019                      |
| LLL  | December 8, 2019 2:15 PM         |
| lll  | Dec 8, 2019 2:15 PM              |
| LLLL | Sunday, December 8, 2019 2:15 PM |
| llll | Sun, Dec 8, 2019 2:15 PM         |
